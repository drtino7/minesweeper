
# minesweeper

The minesweeper is a computer game played on a rectangular grid. The objective of the game is to clear the grid without detonating any mines. The player must uncover the empty squares and hidden mines on the grid. If the player uncovers a mine, they lose the game. If the player uncovers all empty squares without detonating any mines, they win the game. Buscaminas is a game of logic and strategy that requires concentration and skill to win. ## Run Locally

Clone the project

```bash
  git clone https://https://github.com/drtino7/minesweeper/
```

Go to the project directory

```bash
  cd minesweeper
```

Install dependencies

```bash
  pip install pygame
```

execute

```bash
  python3 minesweeper/main.py
```
